## Troubleshooting

### macOS: "CortexIDE is damaged and can't be opened"
If you see this error after downloading the DMG, macOS Gatekeeper has quarantined the app. To fix:

**Option 1 (Recommended):** Run the fix script from the [cortexide-builder repository](https://github.com/OpenCortexIDE/cortexide-builder):
```bash
curl -L https://raw.githubusercontent.com/OpenCortexIDE/cortexide-builder/main/fix_gatekeeper.sh | bash -s /Applications/CortexIDE.app
```

**Option 2:** Manually remove the quarantine attribute:
```bash
sudo xattr -rd com.apple.quarantine /Applications/CortexIDE.app
```

**Option 3:** Right-click the app in Finder and select "Open" (this bypasses Gatekeeper once).
